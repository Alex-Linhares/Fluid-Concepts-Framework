The Graphic User Interface¶

class farg.ui.gui.gui.GUI(controller, args, geometry='810x700+0+0')[source]¶

A tkinter-based interfact for a FARG app.

It sets up three frames: a buttons frame, a central pane, and an interaction widget. Subclasses can override specific bits of it as needed.

HandleAppSpecificFargException(exception)[source]¶

A hook to allow derivative classes a way to handle specific types of exceptions.

If unhandled, the exception should be rethrown.

By default, this does nothing and rethrows the exception.
