Main¶

Class to set up and start running the application based on flags.

class farg.core.main.Main(argv)[source]¶

The Base class for the Main class of an application.

Based on flags, it sets up the appropriate run mode (which start GUIs if needed). It also does a sanity check on flags and creates certain directories if needed.

CreateRunModeInstance()[source]¶

Create a Runmode instance from the flags.

ProcessCustomFlags()[source]¶

Apps can override this to process app-specific flags.

ProcessFlags()[source]¶

Called after flags have been read in.

Run()[source]¶

Start the run.

For Batch mode and SxS, this means running the app several times, whereas for gui mode this means launching a UI.

VerifyLTMPath()[source]¶

Create a directory for ltms unless flag provided. If provided, verify it exists.

VerifyPersistentDirectoryPath()[source]¶

Verify (or create) the persistent directory.

VerifyStatsPath()[source]¶

Create directory for batch stats unless provided. If provided, verify it exists.

VerifyStoppingConditionSanity()[source]¶

Verify that stopping conditions are specified only in modes where they make sense.

application_name = None¶

Name of application. Must be provided by the derivative class.

batch_ui_class¶

Batch UI class to use for running in non-interactive mode. It should be able to handle any questions that may be generated by its codelets. Subclasses of Main can override this, probably with a subclass of its value here.

alias of BatchUI

controller_class¶

The controller runs the show by scheduling codelets to run. Subclasses of Main can override this, probably with a subclass of its value here.

alias of Controller

gui_class¶

GUI class to use for the tkinter GUI. Subclasses of Main can override this, probably with a subclass of its value here.

alias of GUI

input_spec_reader_class = None¶

In batch and sxs modes, the inputs over which to run are specified in a file. These will be converted to flags passed to individual runs. An input reader should be specified for the file to series of flags conversion. These will usually be a subclass of ReadInputSpec.

run_mode = None¶

The mode for the program (gui, batch, sxs, etc). This is an instance of RunMode.

run_mode_batch_class¶

Class to use for running in Batch mode.

alias of RunModeBatch

run_mode_gui_class¶

Class to use for running in GUI mode.

alias of RunModeGUI

run_mode_single_run_class¶

Class to use for running in single mode.

alias of RunModeSingle

run_mode_sxs_class¶

Class to use for running in SxS mode.

alias of RunModeSxS

stopping_condition_fn = None¶

If not none, this is a function is a stopping condition for stopping the app.

stopping_conditions_class¶

A mapping between stopping condition names and their implmentation (which is a funtion that takes a controller and returns a bool).

alias of StoppingConditions
