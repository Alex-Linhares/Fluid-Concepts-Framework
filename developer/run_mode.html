Run Mode — Non Interactive¶

Sets up GUI displaying statistics for batch or SxS modes.

Sets up a window where all inputs and their progress may be tracked.

The inputs are shown in the left panel, and possibly color-coded according to whether the current run is progressing better than the prior run. For SxS, “prior run” is the base side, whereas in the batch mode it is the last time the batch mode was run.

class farg.core.run_mode.non_interactive.RunMultipleTimes(*, input_spec, gui)[source]¶

Class to run the application several times on a set of inputs.

This is the thread that does all the work (except the UI) for batch and SxS modes.

It keeps a pointer to the GUI and updates the stats (owned by the UI).
