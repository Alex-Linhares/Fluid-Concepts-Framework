Controller¶

class farg.controller.Controller(stream_class=<class 'components.stream.Stream'>, routine_codelets_to_add=None, ltm_name=None)[source]¶

The controller of an application (or of a subspace) implements the Step() method, which takes the next action. It marshals the various pieces — coderack, stream, workspace, and so forth.

AddCodelet(family, urgency, **arguments)¶

Adds a codelet to the coderack.

Quit()¶

Gets called when the app is about to quit, in case any cleanup is needed.

RunUptoNSteps(n_steps)¶

Takes upto N steps. In these, it is possible that an answer is found and an exception raised.

Step()¶

Executes the next (stochastically chosen) step in the model.

coderack = None¶

The coderack. Currently, this always has capacity 10.

ltm = None¶

The LTM (if any)

routine_codelets_to_add = None¶

An iterable of three-tuples that names a family, urgency, and probability of addition. At each step, a codelet is added with the said probability, and with this urgency.

steps_taken = None¶

Number of steps taken

stream = None¶

The stream.
