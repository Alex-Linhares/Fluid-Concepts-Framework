Welcome! This is the documentation for PySeqsee 0.2.1a1, last updated June 18, 2012.

PySeqsee is an open-source Python 3 framework for writing programs to solve complex problems that are not amenable to brute-force solutions. The programs created using Pyseqsee have a blackboard architecture and are akin to the computer program Seqsee whose goal is to extend integer sequences in a human-like way. PySeqsee attempts a clean separation of the domain-specific aspects of applications from the domain-independent parts.
