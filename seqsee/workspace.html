Workspace¶

Workspace. Modeled after the workspace in the Perl version.

Warning

No attempt to optimize is being made. Later, with profiling, we can speed
things up.

Specifically, groups are being kept in a set, supergroups-related bookkeeping is not being maintained, and so forth. If profiling reveals these to be issues, these can later be optimized.

GetConflictingGroups(gp)[source]¶

Get a list of groups conflicting with given group.

Groups g1 and g2 conflict if one group is strictly subsumed by the other and yet is not an element of the other. In other words, set(g1.items) overlaps set(g2.items)

Note

This can be sped up if I am keeping track of supergroups.
