Utility Functions¶

Utility functions for Seqsee.

These include:

  • Comparator functions LessThan, LessThanEq, GreaterThan, GreaterThanEq, Exactly, each of which takes a single argument and returns a one-argument function.
  • Toss(x), which returns True with probability x.
apps.seqsee.util.ChooseAboutN(n, choices)[source]¶

Choose in a way that the expected number of choices is n.

Choices is an iterable of 2-tuples, where the second value is the weight.

An example to show how it works: let choices contain 5 things with weights 10, 20, 30 40, and 50 (thus summing to 150), and let n=3. Then we will keep the first item in the output with probability 3 * 10/150 (i.e., 20%).
